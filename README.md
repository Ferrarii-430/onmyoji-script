# 阴阳师脚本 (Onmyoji Script)

仅适用于《阴阳师》桌面版游戏的自动化脚本工具。

## 使用说明

1. 下载最新发行版本的 `yys-script.exe`
2. 双击 `yys-script.exe` 即可启动脚本
3. **重要：** 确保《阴阳师》桌面版处于最前台运行状态

## 注意事项

⚠️ **重要提醒：**
- 目前版本需要保持《阴阳师》桌面版始终在最前台运行
- 暂未测试是否会被游戏检测（"鬼使黑"）
- 请合理使用，遵守游戏规则
- 使用风险自负
